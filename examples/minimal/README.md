# Fluxible Minimal Example

It includes:

- Minimal setup with webpack and babel
- React integration with SSR

## To run

```bash
npm install
npm run build
npm run start
```

Now you can open `http://localhost:8080` in your browser.

## To develop

```bash
npm install
npm run dev
```

Now you can open `http://localhost:8080` in your browser. The server
will reload and the bundle will be rebuild automatically whenever you
change the code.
